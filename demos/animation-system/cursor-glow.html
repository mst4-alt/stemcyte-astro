Cursor Glow Effect

A subtle ambient warmth that follows your cursor across dark sections. Barely noticeable, but it makes everything feel alive.

Light sections stay clean

The glow effect only activates on dark sections. Light backgrounds like this cream (#FAF7F2) remain untouched, providing visual contrast and letting the content breathe.

This approach keeps the effect subtle and purposeful. It enhances dark UI surfaces without overwhelming the page or creating visual noise on lighter sections where it would feel out of place.
